A scratchpad where I work through the [Learn Auth the Hard Way tutorial](https://www.andrew-best.com/posts/learn-auth-the-hard-way-part-one/).

## Running

### Run and configure the Open ID Connect Provider

```
make auth/run
make auth/open
```
